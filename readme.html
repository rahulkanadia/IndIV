How to interpret this Dashboard

1. Live Status Green check means data is fresh (updated < 20 mins ago).
2. ATM Straddle Blocks Shows the cost of the At-The-Money Straddle (Call + Put).

Top row = Prices.
Bottom row = Implied Volatility.
3. Greeks Table Detailed Option Chain.
Gamma/Vega: Risk Metrics.
Delta: Directional bias (Absolute value).
Use Toggle: Switch between Weekly & Monthly chains.
4. Intraday Chart Tracks how IV changes during the day.
Green = Weekly IV
Blue = Monthly IV
